the c3 gallery

This is a collection of collages I created with some friends of mine. It's name is an acronym for cooperatively created collages.

The content of the collages was restricted in one aspect only:
They had to contain at least one picture of the person which whom I created it.

navigation hints

Click on one of the thumbnails to start your tour through the gallery. Each page displays a collage and a navigation bar to the left. The "EyeInTheEar"-icon and the images itself provide links to slightly bigger versions of the collages.

The other symbols in the navigation bar have the following functions:

proceed to the previous/next collage.
go to the gallery index.
displays an info page that constains some information about the collage and the original photos of my friends.
link back to parasitaere-kapazitaeten page.
